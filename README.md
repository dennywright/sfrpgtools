# Starforger Tools App
This is a Starship Generator App for D&D 5e and Dark Matter

## Development
- Setup nodejs
```
nodenv install 18.7.0
nodenv local 18.7.0
```
- Install the requirements:
```
npm install ...
```
- Run the server:
```
node app.js
```
